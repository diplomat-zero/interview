```
数组中占比超过一半的元素称之为主要元素。给定一个整数数组，找到它的主要元素。若没有，返回-1。

示例 1：
输入：[1,2,5,9,5,9,5,5,5]
输出：5
 
示例 2：
输入：[3,2]
输出：-1
 
示例 3：
输入：[2,2,1,1,1,2,2]
输出：2
 
说明：
你有办法在时间复杂度为 O(N)，空间复杂度为 O(1) 内完成吗？
```

```
class Solution {

    /**
     * @param Integer[] $nums
     * @return Integer
     */
    function majorityElement($nums) {
        sort($nums);
        $mid = intval(floor(count($nums) / 2));
        for ($i = 0; $i < count($nums) - $mid; $i++) {
            if ($nums[$i] == $nums[$i + $mid]) {
                return $nums[$mid];
            }
        }

        return -1;
    }
}
```
