```
递归乘法。写一个递归函数，不使用 * 运算符，实现两个正整数的相乘。可以使用加号、减号、位移，但要吝啬一些。

示例1:

 输入：A = 1, B = 10
 输出：10
示例2:

 输入：A = 3, B = 4
 输出：12
提示:

保证乘法范围不会溢出
```

```
class Solution {

    /**
     * @param Integer $A
     * @param Integer $B
     * @return Integer
     */
    function multiply($A, $B) {
        if ($B == 1) {
            return $A;
        }
        if ($B % 2 == 0) {
            return $this->multiply($A << 1, $B >> 1);
        } else {
            return $A + $this->multiply($A << 1, $B >> 1);
        }
    }
}
```
