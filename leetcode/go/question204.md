```
给定整数 n ，返回 所有小于非负整数 n 的质数的数量 。

 

示例 1：

输入：n = 10
输出：4
解释：小于 10 的质数一共有 4 个, 它们是 2, 3, 5, 7 。
示例 2：

输入：n = 0
输出：0
示例 3：

输入：n = 1
输出：0
 

提示：

0 <= n <= 5 * 106
```

```
这是一个古老的筛素数的方法。方法如下：

初始化长度 O(n) 的标记数组，表示这个数组是否为质数。数组初始化所有的数都是质数.
从 2 开始将当前数字的倍数全都标记为合数。标记到根号n时停止即可。

注意每次找当前素数x的倍数时，是从 x的平方 开始的。因为如果x>2，那么2*x肯定被素数2给过滤了，最小未被过滤的肯定是 x的平方
```

```
func countPrimes(n int) int {
    n_map := make([]bool, n)
    for i := 2; i < n; i++ {
        n_map[i] = true
    }
    for i := 2; i * i < n; i++ {
        if n_map[i] == true {
            for j := i * i; j < n; j += i {
                n_map[j] = false
            }
        }
    }
    res := 0
    for _,value := range n_map {
        if value {
            res++
        }
    }
    return res
}
```
