```
给定一个正整数 n ，将其拆分为 k 个 正整数 的和（ k >= 2 ），并使这些整数的乘积最大化。
返回 你可以获得的最大乘积 。

示例 1:
输入: n = 2
输出: 1
解释: 2 = 1 + 1, 1 × 1 = 1。

示例 2:
输入: n = 10
输出: 36
解释: 10 = 3 + 3 + 4, 3 × 3 × 4 = 36。
 
提示:
2 <= n <= 58
```

```
总结一下，可以用数学归纳法证明对任意一个整数最大的拆分形式为 n = 2^p * 3^q (n >= 3, p, q为自然数)，并且q应该尽可能大（理由：如果q减少2，则p要增加3，而2^3 < 3^2，因此应该尽可能增加q）。
```

```
class Solution {

    /**
     * @param Integer $n
     * @return Integer
     */
    function integerBreak($n) {
        if ($n < 4) {
            return $n - 1;
        }
        $dp[2] = 2;
        $dp[3] = 3;
        for ($i = 4; $i <= $n; $i++) {
            $dp[$i] = max($dp[$i - 2] * 2, $dp[$i - 3] * 3);
        }
        return $dp[$n];
    }
}
```
